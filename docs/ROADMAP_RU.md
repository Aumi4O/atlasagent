# Roadmap

## Phase 0 — Knowledge base
Done in this starter:
- owner profile
- businesses
- filters
- prompts
- templates
- logs schema

## Phase 1 — Discovery engine
Goal:
- collect opportunities
- normalize fields
- score fit
- write to log

Output:
- ranked opportunity list
- structured CSV/JSON entries

## Phase 2 — Application packet engine
Goal:
- choose track
- choose resume
- generate intro / cover / short answers
- prepare attachments list

Output:
- ready-to-submit application packet

## Phase 3 — Assisted browser execution
Goal:
- log in / register
- fill forms
- upload docs
- stop before final submit unless approved

Output:
- saved application draft or approval-ready run

## Phase 4 — Controlled autonomous execution
Goal:
- auto-submit on already-approved platform adapters
- checkpoint failures
- avoid duplicate submissions

Output:
- repeatable application pipeline

## Phase 5 — Follow-up loop
Goal:
- track responses
- reminders
- reply drafting
- status updates
