# Application Execution Checklist

1. Read structured vacancy data.
2. Confirm opportunity is allowed.
3. Select positioning track.
4. Select correct resume.
5. Select links:
   - AI Fashion Artists
   - Lead Agents Studio
   - SmartLine
6. Generate tailored intro / cover / short answers.
7. If no platform account exists:
   - create account using env-based credentials
   - stop on verification blockers
8. Fill profile truthfully.
9. Upload files.
10. Answer screening questions.
11. Stop for approval if flow is in assisted mode.
12. Submit only if all checks pass.
13. Write log entry immediately.
14. Save checkpoint or final state.
